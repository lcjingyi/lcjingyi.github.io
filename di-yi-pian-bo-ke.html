一、博客介绍

本博客采用Pelican搭建，纯静态页面，托管于github，主要记录一些学习过程中的记录。

二、搭建过程

  1. 安装依赖
pip install pelican markdown  # markdown 支持 Markdown 语法解析
  1. 初始化博客
mkdir my_static_blog && cd my_static_blog
pelican-quickstart   # 交互式初始化（按提示输入博客名称、作者、URL 等）
  1. 第一篇文章 在 content/ 目录下创建 hello-world.md 文件，内容如下：
Title: xxxx
Date: xxxx-xx-xx xx:xx
Category: xx
Tags: xx, xx

这是用 Pelican 搭建的静态博客，支持 Markdown 语法！
  1. 生成本地文件预览
pelican content  # 生成静态 HTML 到 output/ 目录
pelican --listen  # 启动本地预览服务器

在浏览器查看http://127.0.0.1:8000

  1. 部署到github
# 安装部署插件
pip install ghp-import
# 推送静态文件到 GitHub Pages
ghp-import -n -p output/

之后访问 https://你的GitHub用户名.github.io/ 即可访问博客。

三、自定义主题

在根目录下创建themes文件夹，保存自己的主题样式
